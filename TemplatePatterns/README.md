# 탬플릿 메소드 패턴

## 정의

알고리즘의 각 단계를 정의하며, 서브클래스에서 일부 단계를 구현할 수 있도록 유도하는 패턴 <br/>
추상 부모 클래스와 구상 자식 클래스로 구성되어 있음
